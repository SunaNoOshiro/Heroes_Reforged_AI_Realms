Spectator mode is **out of M5 scope** ( + ). This module is a Phase-4 placeholder that pins the
requirements any future spectator implementation must inherit protections (envelope MAC, handshake binding,
read-only client, visibility precondition, pack signature, build
attestation). See
[`docs/architecture/spectator-mode-requirements.md`](../../docs/architecture/spectator-mode-requirements.md).

**Total Estimate**: pending Phase-4 scoping

satisfies every row in the requirement tables in
[`docs/architecture/spectator-mode-requirements.md`](../../docs/architecture/spectator-mode-requirements.md)
