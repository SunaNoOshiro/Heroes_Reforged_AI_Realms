# Module: Content Editor (M4)

An in-browser creator studio for creating and modifying game content.
No external tooling required — users author, validate, and export
packs without leaving the game.

**Milestone**: M4 — Mods + Content  
**Total Estimate**: ~32 hours  
**Exit Criteria**: A user can create a new unit, set its stats, validate it, and export it as a content pack JSON file — all in the browser.
