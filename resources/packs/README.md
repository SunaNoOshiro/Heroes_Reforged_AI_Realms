# resources/packs

Pack payloads live here once authored content starts shipping.

Canonical layout lives in
[`docs/architecture/pack-contract.md`](../../docs/architecture/pack-contract.md).
One folder under `resources/packs/` equals one pack root with its own
`manifest.json`.

Examples:

- official starter factions
- worlds and scenarios
- balance patch packs
- visual reskin packs
- user-generated exported packs
